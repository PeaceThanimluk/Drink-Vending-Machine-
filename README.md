## Main System
- ระบบเงิน
- ระบบทำเงินด้วยการคลิก
- ระบบซื้อของ

## วิธีการเล่น
1. ดูว่าต้องการสินค้าอะไรในตู้ จากนั้นดูว่าราคาเท่าไร
2. เมื่อทราบราคาแล้ว ให้ทำเงินจนกว่าจะพอกับราคาสินค้า
3. กรอกหมายเลขของสินค้า(อยู่ข้างใต้รูปภาพสินค้า) ที่ปุ่มกรอกเลขที่ต้องการ
4. กดซื้อสินค้า

### ปัญหาที่พบของตัวผมเอง
- เป็นครั้งแรกที่ผมได้ทำโปรเจคผมจึงยังไม่มีความคุ้นชินครับ
- ส่วนของ html ผมสามารถจัดโครงสร้างได้ค่อนข้างโอเค
- แต่ส่วนของ css เพราะผมยังไม่มั่นใจในการเขียน css ทำให้ผมเลยเขียนมั่วๆนิดหน่อย5555
- เนื่องจากผมทำเว็บเป็นของmobileก่อนค่อยเปลี่ยนresponsive เป็นของ desktop เป็นครั้งแรกที่ทำแบบนี้เลยเกิดปัญหา responsive desktop
- เวลาจะเป็น responsive ของ desktop เพื่อให้กลายเป็นแนวนอน ปัญหาที่ตามมาคือ ui ใหญ่เกิน ผมต้องแก้แทบทุกส่วน💀
- ผมจะไปทำความเข้าใจเกี่ยวกับ margin padding max-width ให้มากขึ้นครับ
